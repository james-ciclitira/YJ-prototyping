How Yunojuno works

To begin finding freelancers you first need to create a brief

1. Create a breif

A brief is a summary about the project, this is shown to freelancers to help them decide if they are right for the role.

2. Find a freelancer

Search & shortlist freelancers to ask if they are interested & available to work on the project.

3. Make a booking

Once you’ve found the perfect freelancer you’ll can send a booking request finalising the price and dates.
